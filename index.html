Die Heilbronner Chaos Party -- DHCP 2024

Status of this Memo

This document specifies an Chaos Event protocol for the Chaos community, and requests discussion and suggestions for improvements. Please refer to the current edition of "Die Heilbronner Chaos Party" (DHCP24) for the standardization state and status of this protocol. Distribution of this memo is unlimited.

Abstract

Die Heilbronner Chaos Party (DHCP) is an event organized by Code for Heilbronn (CFHN) for decentralized, distributed, collaborative Chaos and F.U.N.. The event will take place from the 31.10.2024 - 03.11.2024 in Heilbronn. It is intended for people and non-people of all kinds and species. A very important feature are community based contributions such as workshops and talks. DHCP will be used by the World-Wide Chaos community for (at least) this year. This specification defines the protocol referred to as "DHCP24".

1 Introduction

1.1 Purpose

DHCP24 an event organized by Code for Heilbronn e.V. for decentralized, distributed, collaborative Chaos and F.U.N.. DHCP24 is designed to provide all sorts of cultural value. It is intended to provide cognitive challenges through talks and workshops, but also so-called Capture The Flags (CTF).

1.2 Requirements

The key words "MUST", "MUST NOT", "REQUIRED", "SHALL", "SHALL NOT", "SHOULD", "SHOULD NOT", "RECOMMENDED", "MAY", and "OPTIONAL" in this document are to be interpreted as described in RFC 2119 [1]. An attendee is not compliant if they fail to satisfy one or more of the MUST or REQUIRED level requirements for the event it attends. An attendee that satisfies all the MUST or REQUIRED level and all the SHOULD level requirements for its event is said to be "unconditionally compliant"; one that satisfies all the MUST level requirements but not all the SHOULD level requirements for its event is said to be "conditionally compliant."

1.4 Overall Operation

Creatures from various places gather at the event location and date specified in Date & Location to spend time together.
A valid ticket MUST be provided.

Compatibility of creatures with the DE protocol is RECOMMENDED, however other protocols like EN MAY be supported by other participants.
Creatures MUST be excellent to each other.

Detailed information about the event is available in the Wiki.

Updates about the event will be published at @cfhn@chaos.social which you SHOULD follow to avoid important announcments like the ticket sale date.
